# Automated Phishing Link Detector

---

## Project Overview
This Python-based utility automates the identification of malicious URL threats. It was developed to demonstrate how automation can scale security audits by replacing manual inspection with scripted logic.

## Technical Features
* **Pattern Recognition:** Utilizes Regular Expressions (Regex) to identify suspicious domain characteristics.
* **Risk Scoring:** Implements conditional logic to assign a 'Risk Score' based on pre-defined phishing indicators.
* **Security Baseline:** Validates data against a trusted configuration file to minimize false positives.

## Core Competencies
* Python Scripting
* Security Automation
* Regex Design
* Logic-Based Incident Response

---

## Repository Structure
* **detector.py**: Main execution script for URL analysis.
* **config.py**: Trusted domain and keyword configuration.
* **emails.txt**: Test dataset for validation.
